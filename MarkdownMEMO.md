# GitHub Markdown (GFM) 一覧表

| 機能 | 記述方法 | 表示例・備考 |
| :--- | :---: | :--- |
| **見出し** | `# 見出し1` 〜 `###### 見出し6` | # 見出し1 |
| **太字** | `**テキスト**` | **太字** |
| **斜体** | `*テキスト*` | *斜体* |
| **打ち消し線** | `~~テキスト~~` | ~~打ち消し線~~ |
| **箇条書き** | `- 項目` または `* 項目` | ・項目 |
| **番号付き** | `1. 項目` | 1. 項目 |
| **タスクリスト** | `- [ ] 未` `- [x] 済` | [ ] 未 / [x] 済 |
| **インラインコード** | `` `code` `` | `code` |
| **コードブロック** | \`\`\`lang \n code \n \`\`\` | 言語別色付け |
| **リンク** | `[表示名](URL)` | [表示名](URL) |
| **画像** | `![代替テキスト](画像URL)` | 画像の表示 |
| **引用** | `> 引用文` | 枠付きの引用 |
| **水平線** | `---` | 境界線 |
| **テーブル** | `| 項目 | 内容 |` | 表組み |
| **メンション** | `@ユーザー名` | @username |
| **Issue/PR参照** | `#123` | 自動リンク |
| **絵文字** | `:smile:` | 😄 |
| **折りたたみ** | `<details><summary>題名</summary>内容</details>` | クリックで展開 |
